## 1. 为什么付费试用

| 免费 POC 问题 | 付费 Q0 对策 |
|---------------|--------------|
| pilot purgatory | 书面结案 + 90 天上限（R2） |
| 客户无 skin in the game | 低额试用费 + Shadow 报告 |
| 实施边界模糊 | 固定 D1 五项范围（R1） |
| 无法验证转化 | 试用协议含 **转化条款** |

---

## 2. Q0 时间线（≤90 天）

```text
Week 0      准入评估 + 合同（试用费 + 转化价预告）
Week 1–2    只读接入 + Graph draft workshop
Week 2–3    Graph/Rule freeze + DSL 绑定
Week 3–6    Shadow Mode（≥14 天，零生产写）
Week 6–8    UAT 小范围写 + Revert 演示
Week 8–10   灰度 + 培训
Week 10–12  D1 验收签字 + export Package
            → 转化窗口（30 天内转 Starter/Pro）
```

**研发期**（Phase 0–1 内核建设）**不计入** 客户 90 天。

---

## 3. Q0 交付范围（= D1 通病包）

与 08 §3.2 完全一致：

| # | 能力 |
|---|------|
| D1-1 | 跨系统只读查询 |
| D1-2 | Governed 写入 |
| D1-3 | 全链路 Audit |
| D1-4 | Revert 可演示 |
| D1-5 | 每日对账 |

标配 Skill：`skill-work-report-v1`

**不含**：计件、排产、质量全链、WMS 写、OA 自动审批 → 一律 D2

## 4. 收费结构（框架，数字 TBD）

| 项 | 说明 |
|----|------|
| **Q0 试用费** | 低（覆盖 Shadow 期人力）；可抵扣首年订阅 |
| **转化：L0 平台订阅** | Starter / Pro 年费 |
| **转化：L1 D1 Pack** | 含在 Starter |
| **实施尾款** | UAT 通过后支付（若合同分期） |
| **未转化** | 导出 Audit 样本；**不** export 完整 Package（或 export 脱敏版） |

---

## 5. 转化 Gate（试用→正式 License）

全部满足方可激活正式 License：

- [ ] D1 五项验收通过
- [ ] Shadow ≥14 天报告已交付
- [ ] 工厂负责人 D1 签字
- [ ] Implementation Package v1 已 export
- [ ] 首年订阅合同签署

激活 Pack 列表见 [PACK-CATALOG](./模块包商业目录.md) Starter 行。

---

## 6. 未转化处理

| 情况 | 动作 |
|------|------|
| 90 天到期未签字 | 关停写能力；只读可保留 30 天或关停 |
| 客户要求延期 | 最多 **1 次** 30 天延期；须书面原因 |
| 客户要求加范围 | 转 **D2 新合同**，不延长 Q0 |
| 技术不达标 | 平台方延期研发；**不** 占用客户 90 天 |

---

## 7. 灯塔厂（哈森）特殊说明

- 第一家可 **内部转账** 模拟试用费，但 **流程与签字不变**（R5）
- 第二家起严格 Q0→转化，验证 **4–8 周复制**
