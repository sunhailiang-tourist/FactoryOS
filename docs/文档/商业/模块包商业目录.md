## 2. 套餐组合（对外口径）

### Community（引流）

```text
cap-read-l0 + graph-production-query（限量）
+ conn-erp-{vendor}-read（单厂）
+ Platform-L0（审计 30 天）
```

### Starter-A（Path A · 哈森灯塔 · 无 MES）

```text
D1 通病包（§1 中 Starter-A ● 项）
+ conn-erp-{vendor}-write + conn-erp-{vendor}-read + conn-dingtalk
+ impl-package-d1
+ 一次性实施费
```

**写路径**：ERP 报工落账；对账 read-back 走 ERP API。

### Starter（Path B · 单厂 D1 · 有 MES）

```text
D1 通病包（§1 中 Starter ● 项，含 cap-perception-v1 多模态入口）
+ conn-mes-{vendor} + conn-erp-{vendor}-read + conn-{im}
+ impl-package-d1
+ 一次性实施费
```

**写路径**：MES 报工；ERP 只读对账。

### Starter-B-Lite（Path C · 无 ERP · 产品 B Lite）

```text
D1 通病包（无 conn-erp-*）+ conn-mes-builtin + impl-package-d1
+ 编码 workshop 咨询费（必购）
+ 一次性实施费（高于 Overlay Starter）
```

Y1 默认不销售；老板拍板见 [13-应急开发指引](../../准备/2026-06-16/13-产品B-Lite应急开发指引.md)。

### Pro（单厂深化）

```text
Starter 或 Starter-A + graph-order-trace + graph-inventory-read + cap-analysis-l0
+ 对账全量 + 审计 1 年
```

### Enterprise（集团）

```text
Pro + 多 tenant + override-scope + Integration Studio + SLA + 私有化选项
```
