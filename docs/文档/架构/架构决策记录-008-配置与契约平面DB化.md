## 1. 裁定

| 平面 | 真源 | 废止 |
|------|------|------|
| 租户 / Pack / Relation / Override / 快照 | **PostgreSQL Registry** + Studio API | `integration/tenants/*.yaml` 日常编辑 |
| OpenAPI / Schema / CMV / AC | **Contract Registry** publish/frozen | `contracts/` 日常编辑 |
| Graph / Rule / Execution | 现有业务表（不变） | — |
| DDL 表结构 | `src/server/db/migrations/`（Alembic） | UI 改表 |
| 源码 | Git | — |

**实施界面只认 API + DB**；Git YAML/JSON 仅为 export/import 快照或 CI 镜像。

## 2. 部署形态（私有化 + 托管）

- **同一 Registry 模型**；差异在 tenant 数量与 cell 隔离。
- **私有化**：snapshot export · air-gap import · 单 tenant 常见。
- **托管**：`tenant_id`/`cell_id` 全表 · `tenant_pack_entitlements` · 平台 promote contract_set。
