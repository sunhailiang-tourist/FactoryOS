# 规格说明·可观测性

| 版本 | v1.0.0 |
|------|--------|
| 范围 | Core + GIP 统一 trace/log/metric |
| 决策 | ADR-004 · 膨胀期 SLO |

## 1. 原则

- **structlog** JSON 行（已有 ADR-001）
- **OpenTelemetry** span（Phase 1 起接入，export 可 stub）
- 每次 L2 写、每次 Connector op、每次对账 run **必须** 可关联 `exec_id`

## 2. Span 命名

| Span | 属性（必填） |
|------|-------------|
| `execution.run` | tenant_id, verb, graph_id, graph_version, exec_id |
| `connector.op` | tenant_id, pack_id, system, verb, latency_ms, outcome |
| `reconciliation.run` | tenant_id, status, drift_count |
| `mcp.tools.call` | tenant_id, tool_name, plan_id, trace_id（来自 `_meta.traceparent`） |
| `harness.confirm` | tenant_id, user_id, confirmed |

## 3. Connector outcome 枚举

`success` | `timeout` | `legacy_4xx` | `legacy_5xx` | `mapping_error` | `circuit_open`

写入 ExecutionRecord.`connector_trace[]`。

## 4. SLO（Y1 目标）

| 指标 | 目标 |
|------|------|
| API P99（execute） | < 3s（含 Connector） |
| Connector 成功率 | ≥ 99%（排除 Legacy 4xx） |
| 对账 Job 完成 | 每日 06:00 前 |
| Revert 成功率 | ≥ 99.5% |

## 5. 告警

| 条件 | 动作 |
|------|------|
| drift_detected | IM 通知 + Studio 标红 |
| revert_failed | P1 告警 + 人工 runbook |
| circuit_open | Connector 自动降级只读 |

## 6. 日志红线

- **禁止** 日志输出凭证、PII 全量 payload
- Legacy 请求/响应 **采样** 存 OSS（可选，TTL 7d）
