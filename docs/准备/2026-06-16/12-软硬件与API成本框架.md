## 一、成本分类

| 类别 | 记账科目 | 文档 |
|------|----------|------|
| **云基础设施** | 服务器、数据库、缓存、负载均衡、对象存储 | `10` · 用途细分 `15` |
| **研发工具** | Cursor IDE（AI Coding） | 本文 §二 |
| **研发期 API** | 应用 LLM（LiteLLM）+ **多模态**（ASR/OCR/VLM）联调与 staging | 本文 §三 |
| **域名与 SSL** | 主域名注册 + HTTPS 证书采购 | 本文 §四 |
| **可选监控** | ARMS / SLS 等 | 本文 §四 |

**两本账，勿混：**

- **Cursor**：开发人员写 FactoryOS 代码的工具费，不走 LiteLLM。  
- **LiteLLM / 多模态 API**：产品运行时（开发联调、staging、试点）的调用费，计入 **研发成本**。

## 三、研发期 API 成本（LLM + 多模态）

开发阶段 **LiteLLM 与多模态调用统一计入研发成本**（不含生产租户按量计费，生产试点另按 tenant 预算单列）。

### 3.1 应用 LLM（LiteLLM）

| 用途 | 开发期模型建议 | 说明 |
|------|----------------|------|
| 意图 + 填槽 + DSL 计划 | DeepSeek-V3 / 通义 Qwen-Turbo / GPT-4o-mini | 结构化输出，不需最大模型 |
| staging 手测 | Qwen-Plus / GPT-4o | 略提准确率 |

**控费原则**（与 `UX-001` / `AC-BASE-001` 一致）：

- CI / 单元测试：**mock LLM，$0**  
- 仅 staging 人手联调、Shadow 前冒烟：**走真 API**

| 强度 | 月调用量（估） | 月费粗算 |
|------|----------------|----------|
| 克制（mock 为主） | 500～2,000 次 | **¥50～300** |
| 中等（常测 staging） | 5,000～20,000 次 | **¥300～1,500** |

### 3.2 多模态 API（计入研发成本）

感知层 ASR / OCR / VLM 在 **开发联调、UX-001 验收、staging 多模态手测** 期间产生的调用，**全部计入研发成本**。

| 能力 | 典型计费 | 开发期用量（估） | 阶段月费粗算 |
|------|----------|------------------|--------------|
| **ASR** | 按时长 | 数百～数千条测试语音 | **¥100～500** |
| **OCR** | 按张 | 数百张 | **¥50～200** |
| **VLM** | 按张 / token | 开发少测、staging 适量 | **¥100～500** |
| **合计** | — | — | **¥250～1,200 / 月** |

**控费**：CI 用 `ux/` mock 固定 transcript；优先测扫码 fallback（`D-03`）。

### 3.3 研发期 API 小计

| 口径 | 月费（人民币） | 备注 |
|------|----------------|------|
| **底座期 8～11 周（克制）** | **¥350～1,500** | LLM + 多模态合计 |
| **Y1 全年（含灯塔 staging）** | **¥400～2,000** 均摊 | 试点前多模态手测会冲高 |

## 四、域名与 SSL 证书（采购）

与 `10` 一致：**1 个主域名** + 子域名划分（`api.*` 挂 SLB 生产；`gitlab.*` 挂测试 EIP）。钉钉 H5 **必须 HTTPS**。

### 4.1 域名

| 项 | 定版建议 | 首年 / 续费（阿里云粗算） |
|----|----------|---------------------------|
| **主域名** | 1× `.com`（或已备案 `.cn`） | **¥55～85 / 年**（首年促销低，续费约 ¥69～85） |
| **子域名** | `api.`、`gitlab.`、`staging.` | **¥0**（不另购域名） |
| **备案** | `.cn` 或大陆解析 `.com` | **¥0**（工信部流程；无直接官费） |

> 若公司已有集团主域，仅新增子域名解析：**域名采购 ¥0**，SSL 仍按 §4.2 计。

### 4.2 SSL 证书（写入预算的采购方案）

| 方案 | 适用 | 年费粗算 | 是否写入 Y1 预算 |
|------|------|----------|------------------|
| **A · 免费 DV** | 内测 / 早期 staging | **¥0**（阿里云一年期 DV 或 Let's Encrypt） | 备选，不占用预算 |
| **B · 付费 DV 单域名 ×2** | `api.*` + `gitlab.*` 各一张 | **¥1,600～3,600 / 年** | ✓ 可定版 |
| **C · 付费 DV 通配符** | `*.主域名` 覆盖 api / gitlab / staging | **¥2,000～4,500 / 年** | **✓ 推荐定版** |
| **D · OV 企业证书** | 客户合规 / 招投标要求 | **¥3,000～8,000 / 年** | 按需 |

**Y1 定版采购口径（写入立项）**：**方案 C** — 主域名 + **通配符 DV 证书 1 张**，挂 SLB（生产）与测试 ECS（gitlab）复用或分实例部署。

## 六、与生产试点的分界

| 阶段 | Cursor | 研发 API | 云 |
|------|--------|----------|-----|
| Gate 0 底座编码 | ✓ | mock 为主，少量真调 | 测试环境为主 |
| staging / UX-001 手测 | ✓ | **LLM + 多模态计入研发** | 测试 + 生产 ECS |
| 灯塔 Shadow（真实工人） | ✓ | 按 tenant 预算；多模态按量另计 | 生产栈 |

灯塔上线后，若工人真实语音/识图流量增大，多模态 **按调用量递增**，建议在 LiteLLM / 云控制台设 **租户级月度上限**。
