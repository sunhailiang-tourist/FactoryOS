# 闸门零 决策摘要（一页版）

> 日期：2026-06-16 | 状态：**Accepted** | 全文：[ADR-000](./架构决策记录-000-闸门零总决策.md)

---

## 一句话

**卖制造业 AI 执行 OS（Overlay），不换 ERP：终端 0 智多模态 + 内核 Graph/Rule/Revert 双极强大；哈森灯塔 Path A 验证后扩展。文档已齐，可开写 Core 1.0。**

---

## 我们是什么

| 是 | 不是 |
|----|------|
| **双极**：钉钉/H5 **说/拍/扫** + **frozen Graph + Rule + Revert** | 聊天机器人 / ERP 替代品 |
| 接在 ERP/MES **上面** 的编排与入口 | 客户全量业务数仓 |
| 按 Pack 模块化售卖、可复制的 OS | 一次性定制项目 |

**验证路径**：哈森 Path A（ERP+钉钉）→ 书面结案 → 再对外推广。
