# FactoryOS 命名约定

> 版本：**v1.0.0** | 日期：2026-06-16  
> 状态：**Accepted**（W1 前锁定）  
> 关联：ADR-001 §3 · [os_core-public-api](./os_core-public-api.md)

---

## 1. 原则

**见名知意**：目录、Python 包、HTTP 资源、Pack ID 均须从名称直接读出职责；禁止 `packages`、`utils`、`common` 等泛化顶层名（内核目录名为 **`os_core/`**）。

---

## 2. 仓库顶层

| 路径 | 含义 | 变更频率 |
|------|------|----------|
| **`src/server/os_core/`** | 平台内核（L0/L1/L2 引擎），Core v1.0 冻结域 · import: `os_core.*` | 低 |
| **`src/server/api/`** | 后端 HTTP 适配（FastAPI，唯一 deployable · import: `server.api`） | 中 |
| **`src/apps/web-admin/`** | PC 管理端（React + TS） | 中 |
| **`src/apps/h5-worker/`** | 钉钉/企微 H5 壳 | 中 |
| **`src/integration/`** | GIP 外置：catalog · packs · tenants · connector-agent | 高 |
| **`src/tests/`** | AC · integration · workflow · contract pytest | 随 Core |
| **`contracts/`** | 契约 export 镜像（ADR-008 后非编辑真源） | 随 Gate |
| **`docs/文档/`** | 架构 · 规格 · ADR 文档真源 | 随 Gate |

> **废止路径**（harness `check_legacy_paths` 强制，禁止复活）：`src/apps/api/` · `src/os_core/` · `src/db/` · `src/apps/edge-agent/` · `src/server/api/routes/` · `src/server/api/deps.py` · `src/server/api/error_handlers.py`

### 2.1 `src/server/api/` 内部分层（v6-full）

| 路径 | 含义 |
|------|------|
| `main.py` | 薄入口，仅 export `app` |
| `application/` | 应用工厂 + 组装（`assemble.py`） |
| `router/v1/registry.py` | HTTP 路由第一站 · `ROUTER_PROVIDERS` |
| `config/` | settings · logs · status_code · auth/tenant/quota middleware |
| `modules/<domain>/` | 域模块：`controllers/` · `schemas/` · `application/` · `routers.py` |

## 3. `os_core/` 内核模块（10 个）

Python 包名 **snake_case**；uv workspace 成员路径 `os_core/<module>/`。

| 目录 | 职责（见名知意） | 禁止 |
|------|------------------|------|
| `platform_registry` | Contract/Pack/Tenant DB 真源（ADR-008） | 写 Legacy |
| `shared_contracts` | Pydantic 模型、JSON Schema 加载、错误码、常量、DomainEvent | 业务逻辑 |
| `graph_service` | Graph 版本、freeze、checksum、Override 合并 | 写 Legacy |
| `rule_engine` | 角色、条件、动作授权 | 写 Legacy |
| `execution_service` | DSL 执行、Saga、Revert、对账（**唯一写 Legacy 路径**） | 业务 UI |
| `audit_service` | append-only 审计 | 改历史 |
| `agent_orchestrator` | LangGraph Skill FSM → DSL 计划 | 写 Legacy、绕过 Rule |
| `connector_sdk` | Legacy 读写适配（httpx）、Blueprint Runtime、Registry | 业务规则 |
| `license_service` | Pack 授权、Override 生效域 | 执行 DSL |
| `mcp_gateway` | MCP tools/list、tools/call → DslPlan | 直写 Legacy |

**Python import 前缀**：`os_core.<module>`（例：`from os_core.execution_service.api import execute`）。

## 4. 其他命名

| 类别 | 规则 | 示例 |
|------|------|------|
| Pack ID | `kebab-case` + 类型前缀 | `graph-d1-core`、`conn-erp-kingdee-write` |
| DSL 动词 | `SCREAMING_SNAKE` | `WORK_REPORT`、`QUERY_WO` |
| HTTP API | `/v1/<domain>/...` | `/v1/graphs/freeze` |
| Implementation Package API | **保留** `/v1/packages/export` | 指 Pack 快照，非代码目录 |
| 数据库表 | `snake_case` 复数 | `execution_records`、`connector_instances` |
| Alembic revision | `s01_tenants` 等见 ADR-007 | S-01～S-04 |
| Git tag | `core-v1.0.0` | Core Gate 0 |
